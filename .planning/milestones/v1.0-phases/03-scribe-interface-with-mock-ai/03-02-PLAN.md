**ScribeActionMenu.jsx** -- Step 1 of the two-step flow. Renders the action menu with nested submenus and the free prompt input.

Props:
- `onSelect(actionId, label, breadcrumb)`: Called when an action is selected (direct or from submenu)
- `selectedText` (string): Passed through for potential display (not displayed in this component per CONTEXT.md -- no "before/after" comparison)

Structure (per CONTEXT.md decisions):
```
+----------------------------------+
| [check] Correct grammar          |
| [globe] Translate             >  |
| [pen]   Change tone           >  |
| [magic] Improve               >  |
+----------------------------------+
| [input: "Help me write"] [send]  |
+----------------------------------+
```

Implementation details:
- Import `SCRIBE_ACTIONS` from `./scribeActions`
- State: `activeSubmenu` (string|null) -- which top-level action's submenu is open
- Map over `SCRIBE_ACTIONS` to render each top-level item:
  - Use a wrapper div with `position: relative` for each item
  - `onMouseEnter`: if action has children, set `activeSubmenu` to action.id
  - `onMouseLeave`: set `activeSubmenu` to null (on the WRAPPER div, not the ListItem, to avoid flicker -- see Pitfall 2 from RESEARCH.md)
  - Use cozy-ui ListItem with `button` prop, ListItemIcon with `Icon icon={action.icon}`, ListItemText with `primary={action.label}`
  - For items with children: render `Icon icon={RightIcon} size={16}` as chevron indicator
  - For items without children (direct action): `onClick` calls `onSelect(action.id, action.label, action.label)`
- Submenu rendering:
  - When `activeSubmenu === action.id` and action has children:
  - Render a Paper with `position: absolute, left: '100%', top: 0, minWidth: 180, zIndex: 1`
  - Map over action.children to render sub-items as ListItems
  - Each sub-item onClick: `onSelect(child.id, child.label, \`${action.label} > ${child.label}\`)`
  - Sub-items with icons render ListItemIcon; sub-items with `icon: null` skip the icon slot
- Below the action list, render `<Divider />` then `<ScribePromptInput onSubmit={...} />`
  - ScribePromptInput onSubmit: `(prompt) => onSelect('free-prompt', prompt, prompt)`
- Wrap everything in `<Paper style={{ minWidth: 220 }}>`
- Include PropTypes

IMPORTANT -- Submenu flicker prevention (from RESEARCH.md Pitfall 2):
The wrapper div that contains BOTH the parent ListItem AND the submenu Paper must handle onMouseEnter/onMouseLeave. This ensures moving the mouse from the parent item to the submenu does not trigger a leave event.
