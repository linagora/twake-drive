<objective>
Create the pure contract-parsing core of `scribeResponse.js`: `parseScribeResponse(raw, { surface })`, which transforms any raw LLM response string into a normalized `{ discussion, fragments, valid, fellBack, warnings, raw }` object. It runs a tolerant parse pipeline, validates the shape with zero dependencies, and applies a surface-specific fallback when input is non-conforming — never throwing.

This plan owns the parse/validate/fallback pipeline and its security hardening. The marker grammar, MCP helpers, and JSON Schema artifact are added in plan 02 (same file, wave 2). Nothing is wired into the UI in this phase.

Purpose: Everything downstream in v3.1 (prompt plumbing, dev probe, chat/popover render) depends on this module. It is pure, fully unit-testable, and the single home for all validation + fallback logic.
Output: `scribeResponse.js` (parse core) + `scribeResponse.spec.js` (failing-first then green).
</objective>
