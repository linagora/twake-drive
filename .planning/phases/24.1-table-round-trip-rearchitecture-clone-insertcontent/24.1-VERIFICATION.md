### Design Note: Key Link Pattern Divergence

Plan 02 specified key link `Asc.scope.tableCloneIndices` but the implementation uses `Asc.scope.parsedTables`. This is not a gap — the parsedTables structure contains the index information the plan intended `tableCloneIndices` to carry. The implementation is functionally superior (it carries full cell data alongside the index in one structure). The plan's key link pattern `tableCloneIndices` was not written as an exact string requirement, only as a description of intent.
