# Phase v3.2-03 Plan 03: Truncation feedback + i18n Summary

**ChatMessageList now closes the never-silent guarantee: each user turn whose document context was truncated shows a discreet `role=status` info row right below its bubble, extraction failure reuses the ErrorBubble, the default unlimited case stays silent, and the two notice strings are localized across all five locales with zero new design tokens.**

Absent → nothing rendered.
