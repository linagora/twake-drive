**scribeAI.js — Add classifyScribeError function:**

Add a new exported pure function `classifyScribeError(err)` that returns `{ message: string, canRetry: boolean }`:

- AbortError: return `{ message: '', canRetry: false }` (handled upstream, safety catch)
- FetchError with status 401 or 403: return `{ message: 'Authorization error. Please check your Cozy permissions.', canRetry: false }`
- FetchError with status 429: return `{ message: 'Too many requests. Please wait a moment and try again.', canRetry: true }`
- FetchError with status >= 500: return `{ message: 'The AI service is temporarily unavailable. Please try again.', canRetry: true }`
- FetchError with other status: return `{ message: 'Something went wrong. Please try again later.', canRetry: false }`
- TypeError or message containing 'Failed to fetch' or 'Network request failed': return `{ message: 'Network error. Check your connection and try again.', canRetry: true }`
- Error with message 'Empty response from AI': return `{ message: 'No result received. Please try again.', canRetry: true }`
- Default: return `{ message: 'An unexpected error occurred. Please try again.', canRetry: true }`

Check `err.name === 'FetchError'` and `err.status` (not instanceof — it's from a different module).

**ScribePopover.jsx — Wire classification and retry:**

1. Import `classifyScribeError` from scribeAI.
2. Add `lastAction` state: `const [lastAction, setLastAction] = useState(null)` — stores `{ actionId, label, breadcrumb }`.
3. Expand `result` state shape: add `canRetry` field. Init: `{ text: '', breadcrumb: '', error: '', canRetry: false }`. Update all `setResult` calls to include `canRetry: false` for success paths.
4. In `handleActionSelect`, before the try block, store: `setLastAction({ actionId, label, breadcrumb })`.
5. Replace the catch block: instead of the generic error string, call `classifyScribeError(err)` and use the returned `message` and `canRetry`: `setResult({ text: '', breadcrumb, error: classified.message, canRetry: classified.canRetry })`.
6. Add `handleRetry` callback: `const handleRetry = useCallback(() => { if (lastAction) { handleActionSelect(lastAction.actionId, lastAction.label, lastAction.breadcrumb) } }, [lastAction, handleActionSelect])`.
7. In the `useEffect` that resets on open, also reset `lastAction` to null.
8. Update ScribeResultPanel props in JSX: pass `error={result.error}`, `canRetry={result.canRetry}`, `onRetry={handleRetry}` as new props. Change `resultText={result.error || result.text}` to `resultText={result.text}` (error is now a separate prop).
