**ScribeContext.jsx changes:**

1. Add state: `const [panelWidth, setPanelWidthState] = useState(400)` (default 400px, matching current PANEL_WIDTH).
2. Add a stable `setPanelWidth` callback that clamps the value: `Math.min(Math.max(newWidth, 280), window.innerWidth * 0.6)`. Min 280px (usable chat), max 60% viewport (leave room for editor).
3. Expose `panelWidth` and `setPanelWidth` in the context value and useMemo deps.

**ResizeHandle.jsx — new file:**

Create a vertical drag handle component rendered as the leftmost child inside ScribePanel. Implementation:

- Render a `<div>` that is 6px wide, full height, positioned on the left edge of the panel (flexShrink: 0). On hover, show a 2px-wide vertical accent line (centered) using the theme's primary color at 40% opacity. While dragging, accent line at 80% opacity.
- `cursor: col-resize` on the handle.
- On mousedown: attach `mousemove` and `mouseup` listeners to `document` (not the handle — user drags outside).
- On mousemove: compute new width as `window.innerWidth - e.clientX`. Call `setPanelWidth(newWidth)` from context.
- On mouseup: remove listeners, clear dragging state.
- During drag, add `user-select: none` to `document.body.style` to prevent text selection, and restore on mouseup.
- Use `useCallback` for handlers. Clean up listeners on unmount via useEffect return.
- Component signature: `export const ResizeHandle = () => { ... }`, gets `setPanelWidth` from `useScribe()`.
