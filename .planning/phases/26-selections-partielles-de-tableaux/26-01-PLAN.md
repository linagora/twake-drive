**Function 1: `analyzeTableSelection(table, selStart, selEnd)`**

Returns `{ full: boolean, selectedCells: [{r, c}], ambiguous: boolean, reason: string|null }`.

Logic:
1. Get table range via `table.GetRange()`. If no range, return `{ full: false, selectedCells: [], ambiguous: true, reason: "no_range" }`.
2. Get `tblStart = tblRange.GetStartPos()`, `tblEnd = tblRange.GetEndPos()`.
3. If `tblStart >= selStart && tblEnd <= selEnd`, return `{ full: true, selectedCells: [], ambiguous: false, reason: null }` (entire table is selected).
4. Otherwise iterate all rows/cells:
   - For each cell, get its content range: `cell.GetContent()`, get first and last paragraph, get their ranges' `GetStartPos()` and `GetEndPos()`.
   - If content is null, GetElementsCount() === 0, or ranges are null: treat as "empty cell". If the cell's row has at least one other cell fully inside the selection, include the empty cell too (row-level inclusion). Track these for a second pass.
   - Compute `fullyInside = (cStart >= selStart && cEnd <= selEnd)`.
   - Compute `overlaps = (cEnd >= selStart && cStart <= selEnd)`.
   - If `overlaps && !fullyInside`: this is a partial overlap = ambiguous cell. Push to `ambiguousCells` array.
   - If `fullyInside`: push `{r: r, c: c}` to `selectedCells`.
5. After iteration: if `ambiguousCells.length > 0`, return `{ full: false, selectedCells: [], ambiguous: true, reason: "selection_cuts_cell" }`.
6. Second pass for empty cells: for each row that has at least one cell in `selectedCells`, check if any empty cells in that row were skipped. If so, add them to `selectedCells` (empty cells in a selected row should be included).
7. Sort `selectedCells` by r then c.
8. Return `{ full: false, selectedCells: selectedCells, ambiguous: false, reason: null }`.

**Function 2: `extractPartialTableCells(table, selectedCells)`**

Takes the table and the `selectedCells` array from `analyzeTableSelection`. Returns a string of `[CELL:r,c]...[/CELL]` markers for only the selected cells.

Logic is identical to `extractTableCells` but iterates `selectedCells` instead of all rows/cells:
```javascript
function extractPartialTableCells(table, selectedCells) {
  var cellMd = [];
  for (var i = 0; i < selectedCells.length; i++) {
    var sc = selectedCells[i];
    var cell = table.GetCell(sc.r, sc.c);
    if (!cell) continue;
    var content = cell.GetContent();
    var cellText = "";
    var elemCount = content.GetElementsCount();
    for (var e = 0; e < elemCount; e++) {
      var elem = content.GetElement(e);
      if (elem.GetClassType && elem.GetClassType() === "paragraph") {
        if (cellText.length > 0) cellText = cellText + " ";
        cellText = cellText + paragraphToMarkdown(elem);
      }
    }
    cellMd.push("[CELL:" + sc.r + "," + sc.c + "]" + cellText + "[/CELL]");
  }
  return cellMd.join("\n");
}
```
