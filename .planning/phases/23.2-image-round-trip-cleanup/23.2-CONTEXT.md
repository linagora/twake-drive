<decisions>
## Implementation Decisions

### Code cleanup rules
- Remove ALL `console.log("[Scribe]` debug lines added during Phase 23 exploration
- Remove `getInlineDrawings()` wrapper — call `apiRun.GetInlineDrawings()` directly
- Keep the graceful fallback (`if (!apiRun.GetInlineDrawings)`) for when SDK is not patched
- Keep `getDrawingMarker()` for block images — it handles the block vs inline decision
- The `drawingNameMap` approach in `paragraphToMarkdown` is no longer needed — `GetInlineDrawings` returns proper ApiDrawing objects with `GetName()`

### What NOT to change
- The image reinjection code (Copy() approach) — working correctly
- The `SetName("scribe-img-N")` extraction logic — working correctly
- The `MarkdownPreview` badge rendering — working correctly
- The `flattenTokens` image handling — working correctly

### Claude's Discretion
- Exact code structure after cleanup
- Whether to split into smaller functions or keep inline
