**Add stripOoClasses function** (before init, after the helpers section):
```javascript
// Strip OO-internal CSS classes from extracted HTML (from official OO HTML plugin pattern)
function stripOoClasses(html) {
  return html.replace(/class="[a-zA-Z0-9\-:;+"\\/=]*/g, "");
}
```

**Rewrite window.Asc.plugin.init** for the initDataType:"html" path:
- `data` parameter now contains HTML string (not plain text)
- Store class-stripped HTML: `lastSelectedHtml = stripOoClasses(data || "")`
- Call GetSelectedText via executeMethod to get plain text in parallel (needed for SHOW_SCRIBE_BUTTON display and as fallback)
- In the GetSelectedText callback: store `lastSelectedText`, run the existing debounce/button logic
- If GetSelectedText fails or returns empty but we have HTML, extract a text approximation by stripping tags: `data.replace(/<[^>]*>/g, "").replace(/&nbsp;/g, " ").replace(/^\s+|\s+$/g, "")`
