    ```bash
    ## Problem

    `ApiParagraph` provides two disconnected ways to access content:

    | Method | Returns | Includes drawings? | Position info? |
    |--------|---------|---------------------|----------------|
    | `GetElementsCount()` + `GetElement(i)` | Runs and hyperlinks in order | **No** | Yes (index) |
    | `GetAllDrawingObjects()` | All drawings in the paragraph | Yes | **No** |

    There is no way to determine where an inline drawing sits relative to text within its parent run. Inline drawings are children of `ParaRun.Content[]`, but this structure is not exposed through the public API.

    ## Use case

    We are building an AI writing assistant plugin that needs to:
    1. Extract selected text as markdown with image placeholders at correct positions
    2. Send to LLM (which preserves placeholders)
    3. Reinject text with images restored at their new positions

    Step 1 requires knowing the character position of each inline drawing within its run.

    ## Proposed solution

    Add `ApiRun.prototype.GetInlineDrawings()` returning `Array<{drawing: ApiDrawing, position: number}>` where `position` matches the character index in `GetText()`.

    This is purely additive — no changes to existing methods, no breaking changes. ~25 lines of implementation.

    I have a working implementation with tests ready as a PR. Would you be open to reviewing it?

    ## Context

    - OnlyOffice version: 9.3.0-138
    - Use context: Plugin `callCommand` (Document Builder API)
    - Project: [Cozy Cloud](https://cozy.io/) — AI writing assistant plugin
    ISSUE_EOF
    )"
    ```

    ```bash
    Plugins currently cannot determine the position of inline drawings within a text run. \`GetElement(i)\` returns runs/hyperlinks but not drawings (they are children of runs). \`GetAllDrawingObjects()\` returns drawings but without position info.

    This method bridges that gap, enabling plugins to reconstruct mixed content (text interleaved with drawings) in document order.

     */

    \`\`\`

    ### Return value

    Each entry contains:
    - \`drawing\`: \`ApiDrawing\` (or subclass: \`ApiImage\`, \`ApiShape\`, etc.) via \`GetApiDrawing()\` factory
    - \`position\`: Zero-based character index matching \`GetText()\` output

    \`\`\`javascript
    var inlineDrawings = run.GetInlineDrawings();

      result += text.substring(lastPos, pos);

    - **No breaking changes**: New method only
    - **Performance**: Negligible — iterates run content (typically <20 elements), only called when needed
    ```
