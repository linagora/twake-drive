<decisions_made>

- Removed JWT token from frontend DocEditor config — OO 9.x checkJwt runs even with enable=false, and cozy-stack's token field name (`editor` vs `editorConfig`) is incompatible. Disabling token entirely for dev avoids all issues.
- Kept outbox JWT enabled — cozy-stack callback handler requires signed token from OO for save operations.
- Changed cozy-stack Go struct tag from `json:"editor"` to `json:"editorConfig"` — OO 9.x API expects `editorConfig` in the JWT payload. This is a real bug fix.
- Used CSS modules import (`import styles from`) instead of raw class names — cozy-drive webpack config uses CSS modules for .styl files.
- Used `overflow: visible` on Popover PaperProps — needed for submenus to render outside the popover bounds.
- Removed state resets from handleReplace/handleInsert/handleClose — the useEffect on `open` handles reset on next open, avoiding menu flicker.
</decisions_made>
