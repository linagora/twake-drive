<interfaces>
<!-- Public surface this plan consumes. Source: scribeResponse.js (v3.1-01). Use directly. -->

parseScribeResponse(raw, { surface }) returns:
  { discussion: string, fragments: string[], valid: boolean,
    fellBack: boolean, warnings: string[], raw: string }
  - surface 'chat' (default): fallback yields { discussion: raw, fragments: [] }
  - NEVER throws.

serializeAssistantTurnForHistory(parsed) returns string:
  - parsed = { discussion?, fragments? }; returns discussion plus, when fragments
    exist, a compact English "[Previously produced N insertable fragments (truncated):…]"
    note. Never emits full fragment bodies verbatim; never re-emits raw contract JSON.

extractChannelMarkers(text, 'fragment') returns Array of { index, position }:
  - Ordered {{fragment:N}} hits; 0-indexed; REF-safe; read-only. Use this to locate
    markers in the compose helper rather than re-implementing a regex.

FRAGMENT_MARKER_RE strict, REF-safe (matches the {{fragment:N}} literal). Do not re-invent.
</interfaces>
