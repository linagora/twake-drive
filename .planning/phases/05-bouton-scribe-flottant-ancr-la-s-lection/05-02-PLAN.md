**Convert iframe-relative coordinates to host-page coordinates** for the floating button position. The `selectionState` from `useCozyBridge` contains `top` and `left` in OO editor iframe viewport coordinates. Convert them to host-page coordinates using `iframe.getBoundingClientRect()`:

```javascript
const hostPosition = useMemo(() => {
  if (!selectionState) return null

  const iframe = document.getElementsByName(FRAME_EDITOR_NAME)[0]
  if (!iframe) return null

  const rect = iframe.getBoundingClientRect()
  return {
    top: rect.top + selectionState.top,
    left: rect.left + selectionState.left
  }
}, [selectionState])
```

**Trigger-intent approach for the floating button click.** When the floating button is clicked, send a `cozy-bridge:trigger-intent` message to the plugin iframe. The plugin receives this, calls `castIntent("AI_TEXT_EDIT", { text: lastSelectedText })`, which flows back through the normal intent channel, sets `pendingIntent`, and opens the popover. This keeps the architecture clean: all intents originate from the plugin, all responses go back to the plugin.

```javascript
const triggerScribe = useCallback(() => {
  const iframe = document.getElementsByName(FRAME_EDITOR_NAME)[0]
  if (!iframe || !iframe.contentWindow) return
  iframe.contentWindow.postMessage({
    type: 'cozy-bridge:trigger-intent',
    version: 1,
    action: 'AI_TEXT_EDIT'
  }, '*')
}, [])
```

**Plugin code.js -- Add trigger-intent listener:**

Add a message listener in code.js (inside the IIFE, near the existing response listener) that listens for `cozy-bridge:trigger-intent`:

```javascript
// Listen for trigger-intent from Cozy Drive (floating button click)
window.addEventListener("message", function(event) {
  var msg = event.data;
  if (!msg || msg.type !== "cozy-bridge:trigger-intent" || msg.version !== 1) return;
  if (msg.action === "AI_TEXT_EDIT" && lastSelectedText.length > 0) {
    log("Trigger-intent received, casting AI_TEXT_EDIT");
    castIntent("AI_TEXT_EDIT", { text: lastSelectedText });
  }
});
```

```jsx
const ScribePopover = ({ open, selectedText, onReplace, onInsert, onCancel, anchorEl }) => {
  // ... existing state and handlers ...

  const useElementAnchor = anchorEl && anchorEl instanceof Element

  return (
    <Popover
      open={open}
      anchorReference={useElementAnchor ? 'anchorEl' : 'anchorPosition'}
      anchorEl={useElementAnchor ? anchorEl : undefined}
      anchorPosition={useElementAnchor ? undefined : { top: 200, left: typeof window !== 'undefined' ? window.innerWidth / 2 : 500 }}
```
