<objective>
Create the pure placement helper `buildAssistantSegments(discussion, fragments)` that converts the
`{discussion, fragments[]}` contract model into an ordered array of render segments
(`{type:'prose', md}` / `{type:'card', index, raw}`). This replaces the throwaway
`composeAssistantDisplay` text-substitution logic with a segment array a `<FragmentCard>` React node
can be emitted from, while preserving the exact REF-safe placement, orphans-at-end, and out-of-range
strip behavior of the existing code.

Purpose: A `<Card>` is a React element, not text — it cannot be produced by string substitution. This
pure, unit-testable helper isolates all the placement logic (CONTRACT-02 / FRAG-01 / D-04) from the
React render, so the highest-value tests land first.
Output: `assistantSegments.js` + `assistantSegments.spec.js`.
</objective>
